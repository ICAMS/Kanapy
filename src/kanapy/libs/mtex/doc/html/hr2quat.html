hr2quat

arbitrary quaternion q with q * h = r

Description

The method hr2quat defines a rotation q by a crystal direction h and a specimen direction r such that q * h = r

Syntax 

q = hr2quat(h,r)

Input

h Miller or vector3d
r vector3d

Output

q quaternion
